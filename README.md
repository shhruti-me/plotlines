PlotLines - Graph-Based Movie Recommendation System

A full-stack movie recommendation platform that leverages a graph database (Neo4j) to generate personalized and explainable recommendations. Instead of relying on traditional rating-based approaches, the system models users, movies, and metadata as interconnected nodes, enabling relationship-driven discovery through graph traversal.

The platform supports both explicit user input (liked/disliked movies) and natural language preferences, which are processed using NLP techniques to extract relevant entities and dynamically update user profiles. Recommendations are generated by traversing shared attributes such as genres, keywords, and cast, producing relevance-based results with transparent reasoning.

Technologies & Concepts:

Neo4j graph database (graph data modeling & traversal)
FastAPI (REST API design)
React + TypeScript (frontend interface)
Natural Language Processing (spaCy)
Graph-based recommendation algorithms
User preference modeling (LIKES / DISLIKES relationships)
Real-time search with debounced queries
Full-stack system architecture (frontend → API → database)
Explainable recommendation systems
